# Design Direction — Slopify

## Core visual goal
Slopify should look like a polished, generic, AI-enabled SaaS landing page.

This is intentional.

The site should resemble the kind of modern startup / AI tool / micro-SaaS homepage people expect to see:
- clean layout
- gradient accents
- rounded cards
- polished hero section
- feature grid
- testimonials
- pricing cards
- FAQ
- interactive product demo

The joke is not in the visual novelty.
The joke is in the concept, copy, and product behavior.

## Visual tone
- premium
- polished
- generic in a deliberate way
- slightly overconfident
- deadpan
- startup-serious

## Aesthetic references
Broadly reference the style language of:
- modern AI SaaS websites
- typical startup landing pages
- polished commerce/SaaS brands
- clean product marketing pages

Important:
do not directly mimic Shopify branding, logo treatment, colors, or layout structure too closely.

## Desired UI patterns
- large hero headline
- concise subhead
- primary CTA + secondary CTA
- two-panel product demo
- feature cards with icons
- logo wall or fake trust signals
- testimonials
- pricing cards
- FAQ accordion
- clean footer with parody disclaimer

## Typography
- modern sans-serif
- clean and readable
- no experimental display fonts
- strong hierarchy
- headline should feel premium and slightly over-marketed

## Color and styling
- neutral base with AI-style accent gradients
- subtle glow or blur effects are okay
- polished card surfaces
- soft borders
- rounded corners
- restrained use of animation

## Interaction style
- simple, familiar, SaaS-like interactions
- smooth hover states
- polished loading state for text generation
- clear copy-to-clipboard interaction
- crisp transitions, not playful gimmicks

## Product demo expectations
The product demo is the core interaction:
- input text area
- output text area
- generate button
- slop level selector
- mode selector
- example prompt buttons
- loading state
- error/fallback state

## Tone balance
The site should feel:
- believable for 3 seconds
- funny by 10 seconds
- obviously parody by the time someone scrolls

## Avoid
- overly clever design
- maximalist visual jokes
- meme-heavy UI
- complex illustrations
- cluttered dashboards
- dark cyberpunk visuals
- crypto/trading app aesthetics
- anything too close to Shopify’s real branding

## Summary
Build a site that looks exactly like the kind of generic AI SaaS landing page people expect — because that sameness is part of the joke.
